## ğŸ“ Customization

### Update Personal Information
- Edit contact details in `src/components/Contact.jsx`
- Update social links in `src/components/Hero.jsx` and `src/components/Footer.jsx`
- Modify bio in `src/components/About.jsx`

### Add/Edit Projects
- Update projects data in `src/components/Projects.jsx`
- Replace project images with your own

### Change Theme Colors
- Edit colors in `tailwind.config.js`
- Update gradient definitions

### Add Resume
- Place your resume PDF in the `public` folder as `resume.pdf`
